# 쇼핑몰 서버 API

## 1. 프로젝트 소개

### 프로젝트 주제

Golang fiber 웹 프레임워크를 사용한 쇼핑몰 서버 API

### 프로젝트 선정 이유

1. Golang을 사용한 웹 서버 개발 경험 쌓기
2. Golang 사용에 익숙해지기

### 사용 라이브러리

1. fiber v2
2. gorm
3. bcrypt

### fiber 프레임워크 사용 이유

- fiber 웹 프레임 워크는 자바스크립트 웹 프레임워크인 Express의 영향을 받아 만들어졌기 때문에 Node.js를 주로 사용한 경험이 개발에 도움이 됨
